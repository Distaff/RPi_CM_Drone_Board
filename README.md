# Raspberry Pi Compute Module Drone Board (CMDB)

## Description
Used as a lightweight onboard processing unit for AGH Drone Engineering builds. Provides power supply and necessary I/O for Raspberry Pi CM5 module.

IARC HAT Provides programmable ESP32-S3, with SX-1262 for LoRa connectivity, killswitch that disables FC, and ESC/FC connectors.

Weight of both boards is less than 40g.
